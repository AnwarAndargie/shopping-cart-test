# Shopping Cart Test

This repository contains the implementation of a Shopping Cart with features for adding products, calculating totals, and handling sales tax.

## Requirements

To run this code, you need to have the following installed on your machine:
- **Node.js**: (v14 or later recommended)
- **npm**: (Node Package Manager)

## Setup

1. Open your terminal in the project directory.
2. Install the necessary dependencies:

   ```bash
   npm install
   ```

## Running the Code

### Automated Build & Test

A script is provided to automate the build and test process. It installs dependencies, compiles the code, and runs the tests.

```bash
./build.sh
```

### Manual Commands

**Compile TypeScript Code:**
```bash
npm run build
```

**Run Unit Tests:**
```bash
npm test
```
